## 簡介

顏色編輯器，進入網頁時會隨機產生顏色。
可按按鈕 RGB 輸入數值調整顏色，也可透過拖曳調整數值，抑或從下方加減處微調顏色，並即時顯現於畫面當中。
下方問號按鈕則可隨機產生顏色。

## 主要使用技術
* JavaScript
